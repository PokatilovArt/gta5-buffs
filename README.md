# gta5-buffs

Test task for GTA5 RP

## Обработчики баффов

**Серверные**: `onApply`, `onRemove`, `onTick`, `onStackUpdate`.

**Клиент** получает информацию о баффах через события `streamSyncedMetaChange` и `gameEntityCreate` (когда сущность входит в streaming distance). Поэтому актуальная информация о баффах всегда доступна, даже при отсутствии сущности в стриминге. При этом дополнительные события не требуются.
